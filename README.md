一.实验目的

二.内容：利用已学的字符串处理知识编程完成《长恨歌》古诗的整理对齐工作，写出功能函数，并运行。达到如下功能：
 1.每7个汉字加入一个标点符号，奇数时加“，”，偶数时加“。”
 2.允许提供输入参数，统计古诗中某个字或词出现的次数
 考虑操作中可能出现的异常，在程序中设计异常处理程序
